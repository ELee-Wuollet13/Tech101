Engineering Information:

See below for information on software engineering education and career opertunitues in Portland.

Fundimentals of code

  • Coding is writing and formatting specific documents such that a computer or web browser can read them and do what they say.
  • Different languages exist under the umbrella of 'code' and each is used in some scenarios and not others. For example this program uses Javascript, useful becuase it is supported by every modern mainstream web browser.
  • One of the best skills for a new coder is learning how to find answers when you stuck, so for more information check out this Lifehacker video.
  • Networking with others interested in writing code is also an important skill- as with learning any language it easier with a fluent speaker to guide you. In the Portland area try Meetup for events.

Coding schools

  • Epicodus provides a number of fast-paced 'boot camp' options the immerse you in three new languages over the course of five months.
  • Portland Tech Academy offers a wide array of fields of study.
  • pdxcodeguild focuses on the student experience and a well rounded education.
  • Portland State University also offers a coding program.

Professional oppertunities

  • The network you've build while learning to code becomes helpful in finding the job you want in the coding community and many events through meetup and whichever code school you attend will give you chances to meet professionals in the field of your interest, or learn about a new field you hand't previously considered.
  • Having code skills does not negate the need for professionalism. Approach any job in coding as you would any job you really wanted: with a Resume, a collected appearance and respect for all parties involved.
  • A LinkedIn account is recomended as will as a portfolio of you better projects demonstrating whatever skill you feel most proficient in.
